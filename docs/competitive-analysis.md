# FOOKE Competitive Analysis

> Visual scrape + modern competitor references (Jul 2026)

## Target Site: fooke-machines.com

**Company:** FOOKE GmbH — German machine builder (since 1904, Borken/NRW)

**Products:**
- Portal/gantry milling machines (5-axis, long-bed, plate)
- Friction stir welding (FSW) systems
- Additive manufacturing systems

**Industries:** Aerospace, defense, rail, automotive

### Visual / UX Assessment

| Aspect | Current state |
|--------|---------------|
| Aesthetic | Traditional "heavy industrial" — dark navy/grey, black backgrounds |
| Hero | 4-slide auto-rotating carousel, translucent grey text boxes, small blue CTAs |
| Navigation | Cluttered top-nav with long German compound words |
| Layout | Boxy, rigid section stacking (2010s corporate pattern) |
| Cookie UX | Large Usercentrics modal interrupts on load |
| Content strength | Strong use cases (50% lighter, 35% more productive, 43% faster), testimonials, "Made in Germany" story |
| Overall | Content is solid; wrapper feels corporate-dated rather than premium-modern |

### What "modern + catching" looks like in this space

- Cinematic full-bleed hero imagery (often CGI, not stock factory photos)
- Generous whitespace, bold display typography
- Product filter/search bar pinned on hero
- Interactive 3D/spec explorers
- Video-led case studies
- Light + accent-color system instead of all-dark

## Modern Competitors (same industry)

#### 1. Breton — https://breton.it/
Same high-end CNC/composites + aerospace world.

- Cinema-grade CGI hero (humanoid robot on Mars-like landscape)
- Burgundy accent on white, clean centered nav
- Product **filter bar on hero** (Business → Product Category → Filter)

#### 2. Grenzebach — https://www.grenzebach.com/en/markets/friction-stir-welding/machine-portfolio/
Direct FSW-gantry competitor.

- Modern megamenu navigation
- Card-based machine portfolio
- Sustainability report callouts
- Clean white/grey + blue accents
- **Best reference for FOOKE's FSW segment**

#### 3. F. Zimmermann — https://www.zimmermann.de/
Closest direct rival for portal milling (aerospace titanium parts).

- German premium machine-builder positioning
- Cleaner/more refined than FOOKE
- Worth scraping for portal milling product pages

## Key Takeaways for Redesign

1. **Hero:** Replace carousel with single cinematic statement + product finder
2. **Color:** Move from all-dark to light base + bold accent (see Breton burgundy, Grenzebach blue)
3. **Navigation:** Simplify top-level items; use megamenu for product depth
4. **FSW/Additive:** Lead with software/connectivity story (Stirtec model)
5. **Industries:** Card grid with hover video or 3D, not static photo carousels
6. **Proof:** Keep FOOKE's strong metrics (% improvements) but present as animated counters / scroll reveals
